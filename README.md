# IACS Spatial Data Sharing - Technical Guidelines

The purpose of this repository is to host the IACS Technical Guidance documents.

The [issue tracker](https://github.com/geoIACS/technical-guidelines/issues) of this repository can be used to submit and discuss change proposals to the TGs, and to ask any questions relating to the use of the TGs.
The IACS TGs describe detailed implementation aspects and relations with existing standards, technologies, and practices.
